# Web 2 Assignment 1, Movie API
This is a simple CRUD API built with Node.js and the `fs` module.

## How to Run
1. Clone the repo.
2. Run `node server.js`.
3. Open `http://localhost:3000/movies`.

## Features
- GET /movies - Read all movies
- POST /movies - Create a movie
- PUT /movies/:id - Update a movie
- DELETE /movies/:id - Delete a movie
